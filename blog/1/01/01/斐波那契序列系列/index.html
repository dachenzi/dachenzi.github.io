斐波那契序列系列

Jan 1, 0001 · 1 minute read · Comments

{% raw %}

1 打印100以内的斐波那契序列

a, b = 0, 1
print(b)
while True:
    c = a + b
    if c > 100:
        break
    print(c)
    a, b = b, c

2 求斐波那契数列第101项

a, b = 0, 1
count = 1
while True:
    c = a + b
    if count == 100:
        print(c)
    a, b = b, c
    count += 1
